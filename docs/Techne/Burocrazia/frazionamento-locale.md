# Frazionamento dello spazio di un locale

- pratica edilizia

- variazione castale
- diventano 2 unità immobiliari (distinte), quindi nuovo frazionamento, nuova agibilità... servono persone specializzate (ingegneri, architetti) per creare questi documenti
