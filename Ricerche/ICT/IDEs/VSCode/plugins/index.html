Markdown Extension Pack

Setup the plugin named "markdownlint" that is inside this extension pack

  1. go to vscode settings
  2. ctrl + shift + p
  3. write: "Preferences: Open User Settings"
  4. modify plugin configuration
"markdownlint.config": {
    "no-duplicate-header": false,
    "no-hard-tabs": false,
    "no-inline-html": false,
    "single-h1": false,
    "no-bare-urls": false,
        "ul-indent": {
      "indent": 4
    }
}

Setup format on save functionality

  1. ctrl + shift + p
  2. write: "Preferences: Open Settings (UI)"
  3. write: format on save
  4. tick the option
