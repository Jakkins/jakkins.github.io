# How to install Android

## With SDKmanager

```bash
where.exe java
# C:\Users\username\.jdks\semeru-18.0.2\bin\java.exe

# 1. download java
# https://jdk.java.net/21/

# 2. set env var
# JAVA_HOME = C:\Users\username\.jdks\openjdk-21

# 3. set path
# PATH = C:\Users\username\.jdks\openjdk-21\bin
```
